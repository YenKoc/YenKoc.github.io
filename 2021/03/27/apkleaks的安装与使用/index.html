apkleaks的安装与使用

前言

上次开会的时候，听小帅有个可以从app中找接口的一个github项目，最近一直忙资产收集，所以没空看，今晚刚好有空，就clone了一份，并安装了，我习惯用kali了，所以这里安装也是以kali作为主力

安装方法

  • clone一份到本地

  • 进入目录后，输入pip3 install -r requirement.txt

  • 然后就可以使用python apkleads.py -f xx.apk

  • 发现提醒没有jadx的话，要选y，就会自动下载并安装，刚刚去github简介下看了，作者说是apkleaks基于jadx来反编译的

下载后，就会发现同目录下，有一个jadx文件夹了，至此安装完成

使用方法

python apkleaks.py -f xxx.apk

然后就开始执行扫描，会将ip和端口都提取出来，不过我实测还是不够准确的，如果只找接口的话，怎么会这么多其他的东西扫出来呢。。我也感觉奇怪，我建议在这句命令后-o xx.txt, 有些app还是挺大的，结果估计会很多，找个文件保存比较合适
